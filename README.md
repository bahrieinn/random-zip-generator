Random Zip Generator
====================

Setup
====================
1. Clone the repo
2. cd into directory

Usage
====================
1. `ruby random_zip.rb [state_abbreviation]` with any state abbreviation like 'CA', 'NY', etc.
2. Be happy

Output
====================
The response will look something like:
```
> ruby random_zip.rb MN
Minnesota Zip Codes
56063 - Madison Lake
```
